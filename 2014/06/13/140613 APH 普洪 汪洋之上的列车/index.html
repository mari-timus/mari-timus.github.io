汪洋之上的列车

文 / 鹿奂

{ null }

据说，等一个与自己不相爱的人，就像在机场等一艘船。不过，匈牙
利小姐伊丽莎白·海德薇莉总是说，她遇见“那个正确的人”，就是在汪
洋之中的一列火车上。

具体时间，是那一年的正当中，六月的月末。

{ eins }

盛夏的萨尔茨堡，是伊丽莎白梦想了很久的旅行目的地：绿荫掩映中
的尖塔教堂，十字路口喷泉上方的微型彩虹，阿尔卑斯山与阿特湖。

作为莫扎特和卡拉扬共同的故乡，萨尔茨堡会抓紧每一个短暂的夏天
筹办音乐节，这理由几乎是显而易见地漂浮在城市上空的每一缕空气里：
那些萦绕不散的音乐声，就像是海洋中的漩涡。

伊丽莎白自出生起的二十多个年头，都是在布达佩斯度过的。从这个
角度来说，棕色头发的小姐算是个不折不扣的匈牙利人。把音乐认定为了
要奋斗一生的事业之后，伊丽莎白就在心中的“朝圣地列表”中增添了“萨
尔茨堡”一项。

从布达佩斯乘上的火车，半途改主意停下了，她，连同行李被扔在了

多瑙河遭遇了百年一遇的大洪水，萨尔茨堡已经可以划船了。

匈牙利小姐感到一阵令人绝望的晕眩，差点失去平衡，一屁股跌坐在

去他妈的鬼天气。有着一头优美长发的漂亮小姐在心中非常不淑女地
